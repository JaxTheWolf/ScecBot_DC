Commands

delPC1/2

Use this command to delete your existing configuration. 
Usage: *delPC1, *delPC2

setPC1/2

Use this command to set your configuration. The bot will ask you your parts list, and will automatically put them together.
Usage: *setPC1, *setPC2

PC1/2

Use this command to see someone's configuration.
Usage: *PC1 @User#0000, *PC2 @User#0000

credits

Use this command to see the creators of this bot.
Usage: *credits

help

Use this command to display a list of available commands, or detailed information for a specified command.
Usage: *help, *help [command]

Avatar

Use this command to display someone's avatar on discord.
Usage: *avatar @user#0000, *avatar user

reply

Use this command to test if the bot is working as intended. Bot will answer: "It's working!"
Usage: *reply

prefix

Use this to show or set prefix.
Usage: *prefix -, *prefix

ping

Use this to receive bot's ping to Discord server.
Usage: *ping

groups

Use this to receive a list of all command groups.
Usage: *groups

enable

Use this to enable a command or a command group.
Usage: *enable main, *enable ping

disable

Use this to disable a command or a command group.
Usage: *disable main, *disable ping

reload

Use this command to reload a command or a command group.
Usage: *reload main, *reload ping

load

Use this command to load a command or a command group
Usage: *load main, *load ping

unload

Use this command to load a command or a command group.
Usage: *unload main, *unload ping

Invite

Use this command to invite SpecBot to your server.
Usage: *invite

embed

Use this command to test if embeds are bugged out etc.
Usage: *embed [text]

say

Use this command to make the bot say anything.
Usage: *say [text]
