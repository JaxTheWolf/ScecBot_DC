Commands

delPC1/2

Use this command to delete your existing configuration. 
Usage: *delPC1, *delPC2

setPC1/2

Use this command to set your configuration. The bot will ask you your parts list, and will automatically put them together.
Usage: *setPC1, *setPC2

PC1/2

Use this command to see someone's configuration.
Usage: *PC1 @User#0000, *PC2 @User#0000

credits

Use this command to see the creators of this bot.
Usage: *credits
Aliases: *creds, *authors

help

Use this command to display a list of available commands, or detailed information for a specified command.
Usage: *help, *help [command]

avatar

Use this command to display someone's avatar on discord.
Usage: *avatar @user#0000, *avatar user
Aliases: *pfp

reply

Use this command to test if the bot is working as intended. Bot will answer: "It's working!"
Usage: *reply

prefix

Use this to show or set prefix.
Usage: *prefix -, *prefix

ping

Use this to receive bot's ping to Discord server.
Usage: *ping

groups

Use this to receive a list of all command groups.
Usage: *groups

enable

Use this to enable a command or a command group.
Usage: *enable main, *enable ping

disable

Use this to disable a command or a command group.
Usage: *disable main, *disable ping

reload

Use this command to reload a command or a command group.
Usage: *reload main, *reload ping

load

Use this command to load a command or a command group
Usage: *load main, *load ping

unload

Use this command to load a command or a command group.
Usage: *unload main, *unload ping

invite

Use this command to invite SpecBot to your server.
Usage: *invite

embed

Use this command to test if embeds are bugged out etc.
Usage: *embed [text]

say

Use this command to make the bot say anything.
Usage: *say [text]
Aliases: *repeat, *msg

announce

Announces message to all servers the bot is in. Only @oko123#8509 and @The76i#1234 can use this, so owner-only.
Aliases: *ann, *announcement

editPC1

Use this command to edit your current configuration.
Usage: *editpc1 CPU i5-8600k, *editpc2 CPU i5-8600k
Aliases: *edit1, *edit2

suggest

Use this command to suggest a feature for SpecBot.
Usage: *suggest
Aliases: *sugg, *featurereq, *request, *suggestion

report

Use this command to report someone.
Usage: *report @user#0000 offensive memes
Aliases: *repo, *specbotreport, *sprepo, *spreport
Answers for confirmation: True, False
ABUSE OF THIS COMMAND CAN RESULT IN A PUNISHMENT!

donate

Aliases: *contribute, *support

update

This command is used to update the bot. OWNER ONLY!
Usage: *update

restart

This command is used to restart the bot. OWNER ONLY! Currently Linux-only.
Usage: *restart

guildinfo

Use this command to get info of current server.
Usage: *guildinfo
Aliases: *ginfo, *guild_info, *g_info

uptime

Use this command to get current uptime of the bot.
Usage: *uptime

userinfo

Use this command to get user's info.
Usage: *userinfo, *userinfo @user#0000
Aliases: *uinfo, *user_info, *u_info

status

Use this command to get status of the bot. OWNER ONLY! Linux-only.
Usage: *status
